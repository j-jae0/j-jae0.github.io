# <span class="half_HL">1. 색상 종류</span>
**matplotlib.pyplot 라이브러리**를 로드한 후, 아래 코드와 같이 작성하면 사용가능한 색상 종류(이름)을 확인할 수 있다.
```python
import matplotlib.pyplot as plt

print(plt.colormaps())
```

# <span class="half_HL">2. 색상 확인</span>

**목적** : cmap 종류별로 어떤 색조합을 가지는지 확인한다.
